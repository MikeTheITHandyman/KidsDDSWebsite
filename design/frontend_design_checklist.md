Front-end Design Skill Invocation
================================

This file documents the front-end design checklist to invoke before writing any front-end code.

Principles:
- Mobile-first, accessible, and SEO-first layouts
- Use semantic HTML elements (header, main, article, section, nav, footer)
- Color palette: soft blues/greens with warm neutrals and 10-15% energetic accent
- Typography: legible sizes for small screens, scalable via CSS variables
- Images: use next/image, descriptive alt text, optimized sizes
- Performance: minimize render-blocking CSS, prefer server components for static content
- Content: ensure every page has title, meta description, and OpenGraph tags

Deliverables before coding:
- Wireframe sketch of layout (hero, CTAs, nav, footer)
- List of reusable components (Button, Card, SiteNav, SEOHead)
- Accessibility checks (contrast, keyboard nav, landmarks)
- SEO checklist per page (title, meta desc, canonical, structured data plan)
